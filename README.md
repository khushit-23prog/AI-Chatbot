# TASK 3: AI CHATBOT WITH GUI

The AI ChatBot is a Python-based application
developed using the Tkinter library to provide an
interactive conversational experience through a
graphical user interface. The chatbot is designed
to respond to user queries using predefined
responses and keyword-based matching techniques.

The application can perform various tasks such as
greeting users, displaying the current date and
time, sharing motivational messages, providing
quotes and facts, telling jokes, and performing
basic mathematical calculations. The chatbot also
stores conversation history, allowing users to
review previous interactions.

This project demonstrates the practical
implementation of Python programming concepts,
including GUI development, event handling, file
management, conditional statements, and automated
response generation. It provides a simple yet
effective example of chatbot development and
showcases how Python can be used to build
interactive desktop applications.
